Docker Commands

Getting Started

$ docker --version

Check the installed Docker version.

$ docker pull

Pull an image from Docker Hub.

Image Management

$ docker images

List all Docker images on your system.

$ docker rmi

Remove a Docker image.

$ docker build -t

Build a Docker image from a Dockerfile.

Container Management

$ docker run -d -p :

Run a container in detached mode and map ports.

$ docker ps

List running containers.

$ docker stop

Stop a running container.

$ docker exec -it  /bin/bash

Access a running container's shell.

Kubernetes Commands

Getting Started

$ kubectl version

Check the installed kubectl version and cluster information.

$ kubectl cluster-info

Get information about the Kubernetes cluster.

Pod Management

$ kubectl get pods

List all Pods in the default namespace.

$ kubectl describe pod

Get detailed information about a specific Pod.

$ kubectl delete pod

Delete a Pod.

Deployment Management

$ kubectl get deployments

List all Deployments in the default namespace.

$ kubectl create -f .yaml

Create resources defined in a YAML file.

$ kubectl apply -f .yaml

Apply changes to resources defined in a YAML file.

$ kubectl rollout status deployment/

Check the rollout status of a Deployment.
